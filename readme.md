## vnumper-check
### A Javascript Library for number manipulation
### - By Ankit Choudhary

# Example-1
```
const vnumber_check = require("vnumber-check")
const input = 12
const output = vnumber_check.isPerfect(12)
console.log(output)

* *****
output : 
false
```
